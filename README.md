## Project Overview
Hybrid Cloud connecting on-premise network (GNS3) to cloud (Docker) via IPsec tunnel.

## Network Topology
- **On-Premise:** 172.16.1.0/24 (Ubuntu Server)
- **Cloud:** 10.0.0.0/24 (Docker container)
- **IPsec Tunnel:** Between Cisco router and StrongSwan gateway
